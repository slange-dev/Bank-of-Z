# Tutorials
After installing and verifying Bank of Z, use the tutorials in this section to explore common development, deployment, and modernization workflows.

The tutorials build on one another and demonstrate how application changes move from source code to deployment within a hybrid IBM Z environment.

By completing these tutorials, learn how to:

- Build and deploy Bank of Z application components
- Modify and validate application code
- Deploy changes to CICS and IMS environments
- Use modern IBM Z development workflows and tooling
- Verify application behavior after deployment

## Available Tutorials

| Tutorial | Description |
|-----------|-------------|
| [**Deploy Bank of Z**](deploy-bank-of-z.html) | Watch a complete first-time deployment of Bank of Z. |
| [**CICS Enhancement Scenario**](cics-enhancement-scenario.html) | Implement and deploy an enhancement to a CICS-based application component. |
| [**CICS Debugging Scenario**](debug-cics-transaction.html) | Debug a CICS transaction in VS Code IDE. |
