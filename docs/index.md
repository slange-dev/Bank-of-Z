Welcome to the Bank of Z documentation. Use this guide to explore the application architecture, set up a development environment, follow hands-on tutorials, and learn modern IBM Z development workflows.

The documentation includes installation guidance, development workflows, architecture information, tutorials, reference material, and troubleshooting resources.

![Sample Online Banking Application](docs/Image/sample-banking-app.jpg)

## What You Can Learn

Bank of Z provides guided scenarios that demonstrate common IBM Z development and modernization activities, including:

- Install and configure the Bank of Z application environment
- Build and deploy the application
- Modify CICS and Db2 applications
- Modify IMS TM and IMS DB applications
- Add functional enhancements
- Investigate application performance issues
- Explore hybrid application architectures
